# npy

a collection of packages made for personal use — existing solutions didn't quite fit, so these were built instead.
feel free to use, contribute, or leave feedback if you find them useful :>

> [!WARNING]
> everything here is a work in progress and built for personal purposes.
> breaking changes may happen at any time without prior notice.

## packages

- [`@npy/fetch`](./packages/fetch) — a low-level http client built on raw tcp sockets
- [`@npy/proxy-kit`](./packages/proxy-kit) — proxy uri parsing, stringification and connection utilities
